Getting started

Here you will find the living documentation for the different components that make up and should make up any Manitou website.

This centralized style guide will help to promote visual and functional consistency, to guarantee the same great experience accross our websites and to guide any web designer or developer to build a new platform.

To use it, load its dependencies (see "Scaffolding" below) and the minified css of the framework, and adapt the html code of the framework components you want. Or simply use it with your favorite package manager such as Bower.
