Getting started

Here you will find the living documentation for the different components that make up and should make up any Manitou website.

This centralized style guide will help to promote visual and functional consistency, to guarantee the same great experience accross our websites and to guide any web designer or developer to build a new platform.

To use it, load its dependencies (see "Scaffolding" below) and the minified css of the framework, and adapt the html code of the framework components you want. Or simply use it with your favorite package manager such as Bower.

Scaffolding

The Manitou web framework is build on top of different tools that help its develomment :

  • the front-end framework Bootstrap 4 that brings us a responsive and mobile first grid system and various utilities and pre-built components. Use its own documentation to enhance your website ;
  • the Font Awesome library that gives us a scalable vector icons we can manage as a font. Use its cheatsheet to find the icon you ar elooking for ;
  • some specific components are also build on top of the javascript framework Angular. In the framework, we specifiy in the component's doc when it is the case.
