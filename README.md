Manitou Web Framework
=====================

Manitou Web Framework is build on top of [Twitter Bootstrap](http://getbootstrap.com) in [Less](http://lesscss.org/) (a CSS pre-processor).

## Assets

- **Bootstrap**
- **Less**
- **Font Awesome**
- **Respond.JS** (IE8)
- **html5shiv** (IE8)
- **Grunt** (soon)

## How to build
- get Node.js (http://nodejs.org/)

```
$ npm install
$ grunt
```
