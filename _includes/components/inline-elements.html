Inline elements

Inline elements are used to create hierarchy in the informations.

These elements are visually different from each other, but are not to be used in an esthetic way for all that. They create a logical hierarchy that has to make sense : a <strong> element for instance is not a bold element, it is rather a more important element that is then visually distinguished.
