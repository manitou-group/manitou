Scaffolding

The Manitou web framework is build on top of different tools that help its develomment :

  • the front-end framework Bootstrap 4 that brings us a responsive and mobile first grid system and various utilities and pre-built components. Use its own documentation to enhance your website ;
  • the Font Awesome library that gives us a scalable vector icons we can manage as a font. Use its cheatsheet to find the icon you ar elooking for ;
  • some specific components are also build on top of the javascript framework Angular. In the framework, we specifiy in the component's doc when it is the case.
