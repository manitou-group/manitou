Navigations

Aside Vertical Nav

These blocks can be used as a vertical navigation in a <aside> column.

Open by default the first one with a .in class in the collapse element to suggest the interaction

The .dropdown-toggle defines the element to click to collapse the block. Use its data-target to precise wich block must be collapse by refering its id.
